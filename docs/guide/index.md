# Quae-魁

Quae（发音为/kwei/）中文名为魁，是一款新拟物风格的用于快速构建小型网站的轻量化模板。前端使用 `Vue3` `vite4` `Element-Plus` 开发，后端使用 `SpringCloud` 开发。它能够帮助你快速构建部署一个包含注册用户端、非注册用户端和管理端的 Web 应用程序。
